O Meu Primeiro Teste de Conversão

Este é um ficheiro de exemplo para testar o Mini-Projeto 1. Se estiveres a ler isto num browser com formatação, significa que o teu script funcionou!

Coisas que estou a aprender

Aqui está uma lista do que este projeto ensina: 1. Como ler ficheiros com `fs.readFile`. 2. Como ler argumentos da linha de comandos (`process.argv`). 3. Manipulação de strings básica.

Um pouco de estilo

É importante testar se o texto fica em negrito, em itálico e nos nos dois, negrito e itálico!
